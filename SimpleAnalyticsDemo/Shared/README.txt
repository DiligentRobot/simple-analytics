This demo project is a SwiftUI universal app that requires iOS 14 or macOS 11, and Xcode 12.

For demos that work in earlier OS and Xcode versions, please see the SimpleAnalyticsDemo-UIKit and SimpleAnalyticsDemo-AppKit
projects.

For an example of using SimpleAnalytics in an Objective-C based application, see the SimpleAnalyticDemo-ObjC project.
